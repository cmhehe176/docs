🐳 Hướng dẫn cài đặt Docker trên Ubuntu 22.04 ​

🔥 1. Cập nhật hệ thống ​

zsh
sudo apt update -y && sudo apt upgrade -y

🔥 2. Cài đặt Docker ​

📌 Cài đặt các gói cần thiết ​

zsh
sudo apt install apt-transport-https ca-certificates curl software-properties-common -y

🔑 Thêm khóa GPG của kho lưu trữ Docker ​

zsh
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

📦 Thêm kho lưu trữ Docker ​

zsh
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

🔄 Cập nhật packages và kiểm tra phiên bản Docker từ kho lưu trữ chính thức. ​

zsh
sudo apt update
sudo apt-cache policy docker-ce

🔥 3. Cấu hình quyền Sudo cho User sử dụng Docker ​

⚠️ Docker Daemon chỉ được thực thi với quyền root theo mặc định. Nếu muốn sử dụng Docker mà không cần sudo, hãy thêm user của bạn vào nhóm Docker.

zsh
sudo usermod -aG docker $(whoami)

🔎 Kiểm tra user hiện tại ​

zsh
whoami

🔄 Khởi động lại hệ thống để áp dụng thay đổi, sau đó kiểm tra bằng lệnh:

zsh
docker ps

🎯 Nếu không có lỗi, bạn đã cấu hình thành công!

🔥 4. Docker với WSL (Chỉ dành cho người dùng Windows Subsystem for Linux) ​

📝 WSL là trình giả lập Linux trên Windows, do đó Docker không tự động khởi động như trên Linux thực sự.

⚡ Giải pháp 1: Bật Docker bằng cơm ( cứ lúc nào tắt đi bật lại terminal thì sẽ phải bật lại bằng cơm) ​

zsh
dockerd

🎯 Nếu hiển thị như sau, Docker đã chạy thành công:

zsh
➜  ~ dockerd
INFO[2025-02-25T20:32:33.946336009+07:00] Starting up

⚡ Giải pháp 2: Cấu hình để Docker tự khởi động trong shell ​

🔧 Mở file sudoers để cấp quyền tự động chạy Docker mà không cần nhập mật khẩu:

zsh
sudo visudo

📌 Thêm dòng sau vào cuối file (thay ndmc thành username là user của bạn):

zsh
ndmc ALL=(ALL) NOPASSWD: /usr/bin/dockerd

🔄 Cấu hình tự động chạy Docker khi mở terminal:

  • Đây là lệnh dành cho zsh shell, nếu bạn dùng bash thì đổi .zshrc thành .bashrc
zsh
echo '# Start Docker daemon automatically when logging in if not running.' >> ~/.zshrc
echo 'RUNNING=`ps aux | grep dockerd | grep -v grep`' >> ~/.zshrc
echo 'if [ -z "$RUNNING" ]; then' >> ~/.zshrc
echo '    sudo dockerd > /dev/null 2>&1 &' >> ~/.zshrc
echo '    disown' >> ~/.zshrc
echo 'fi' >> ~/.zshrc

🎯 Từ giờ, Docker sẽ tự động chạy khi bạn mở terminal! 🎉
